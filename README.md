# API Testing Project

This project shows my hands-on practice with API testing using Postman. I worked with REST APIs by sending requests, validating responses, and using environment variables to manage endpoints and test data.

## Overview
Tested API endpoints by executing requests, validating response status codes, and reviewing JSON data for accuracy and expected behavior.

## Tools Used
- Postman
- JSON

## What I Did
- Sent API requests and validated response status codes
- Verified JSON response structure and required fields
- Used Postman environment variables to manage base URLs and test data
- Performed positive and negative testing on API endpoints

## How to Use
1. Import the Postman collection
2. Import the environment file
3. Select the environment in Postman
4. Run the requests and review responses

## Notes
This project reflects my practical experience with API testing and basic QA workflows.
